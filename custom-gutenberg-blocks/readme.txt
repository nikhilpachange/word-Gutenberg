== Description ==
This plugin adds a custom Gutenberg block to WordPress.

== Installation ==
1. Upload the `custom-gutenberg-blocks` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Use the Gutenberg editor to add the "Custom Heading Block" to your posts/pages.
